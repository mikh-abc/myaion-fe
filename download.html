aiDPSMeter - Айон ДПС Метр

aiDPSMeter — это сетевой информационный инструмент для Aion. aiDPSMeter показывает урон, лечение, и скорость анимации, ход PvP-инстансов со списком участников, информацию о персонажах и текущих серверах онлайн. Мы рекомендуем использовать библиотеку Npcap для сбора информации Aion из сети.

Системные требования: Windows Vista, Windows 7, Windows 8, Windows 10, Windows 11

Инструкция:

  1. Скачать и установить Npcap
  2. Перезагрузить Windows
  3. Скачать версию 1.5: msi file
  4. Запустить aiDPSMeter до входа на игровой сервер
  5. В случае нескольких доступных интерфейсов, убедитесь, что выбраный интерфейс используется для подключения к Интернету

Чтобы изменить язык данных Aion используйте L10N Tool.
